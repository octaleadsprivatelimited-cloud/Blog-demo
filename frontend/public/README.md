# Public Assets

Place your demo video file here:

- **demo-video.mp4** - Main video file (recommended)
- **demo-video.webm** - Alternative format for better browser compatibility (optional)
- **demo-video-poster.jpg** - Thumbnail/poster image shown before video loads (optional)

## Recommended Video Settings:
- Format: MP4 (H.264 codec)
- Resolution: 1920x1080 (Full HD) or 1280x720 (HD)
- Aspect Ratio: 16:9
- Duration: 30-60 seconds recommended
- File Size: Keep under 10MB for best performance

## Using External Video (YouTube/Vimeo):
If you prefer to use YouTube or Vimeo, update the video element in `src/pages/Home.jsx` to use an iframe instead.
